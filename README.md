# Looty产品方案
## 一、产品概述
- 产品名称：Looty
- Slogan: Looty, Make games easy.
- 产品定位：专为独立游戏开发者设计的**零后期**AI 2D资产生成工具
- 核心价值：将「生图→PS抠图→导入引擎」流程，简化为「文字输入→直接获取透明PNG游戏素材」一键操作

## 二、目标用户与核心痛点
### 目标用户
无专职美术的独立游戏开发者、游戏策划、Hackathon参赛团队

### 核心痛点
1. 常规AI出图自带背景，无法直接用作游戏精灵图
2. 画风难以把控，达不到2D游戏清晰边缘、纯色底的制作标准
3. 装备迭代生成难以保留原有轮廓形态

## 三、功能需求范围(MVP Scope)
> 注：开发周期2天，按优先级执行，P0必做，P1按需选择开发

### 1. 前端表单与交互（P0）
- 文本输入框：录入道具物品描述
- 画风下拉选择器：美漫粗线矢量风、复古2D像素风
- 一键生成按钮：触发素材生成流程
- 结果展示区：呈现透明底PNG图，附带下载按钮

### 2. 后台Prompt自动封装引擎
无需用户编写提示词，后台自动组合生成工业级指令
组合公式：用户输入内容+画风关键词+白底独立道具约束条件

### 3. AI生图+自动去背闭环
1. 封装指令调用即梦1.5生图接口
2. 后台调用rembg库自动去除图片背景，生成透明通道PNG
3. 成品可直接导入Unity等游戏引擎使用

### 4. 装备进化图生图功能
已有素材支持二次编辑，输入特效描述，以原图为基底生成同轮廓进阶装备，生成后自动完成去背景处理

## 四、界面布局示意图
基于Streamlit/Gradio搭建极简界面
```
======================================================
🪓 Looty | Make games easy. (Header)
======================================================
[左侧控制面板区]                     | [右侧工作区：资产结果]
                                  |
1. 你需要什么装备？                 | 🔄 生成中...(进度条)
   [ 文本输入框：例如"魔法大剑" ]     | 
                                  | 
2. 选择素材画风                     | ✅ 生成完毕！
   [ 下拉菜单：美漫矢量 / 像素风 ]    |
                                  | 🖼️ [展示去背后的透明图片]
3. 🚀 [ 生成游戏资产 (Button) ]     | 
                                  | ⬇️ [ 下载透明 PNG 素材 ]
----------------------------------|
(P1扩展区：装备升级室)              |
➕ 升级此装备：[输入"变为冰属性"]    | 🖼️ [展示升级后的新图片]
✨ [ 进化 (Button) ]               |
======================================================
```

## 五、技术栈
- 前端：Streamlit（Python开发）
- 后端：Python
- AI生图接口：即梦1.5 API
- 图像处理：rembg、Pillow
